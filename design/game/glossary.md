# Glossary（術語表）

> **本文件是設計文件的統一術語對照表。**
> Agent 在閱讀或撰寫 `design/game/` 下任何文件時，應優先參考本表以確保用語一致。

---

## 核心概念

| 術語 | 定義 |
| :--- | :--- |
| **玩家** | 透過連線加入的人或是房主自己。 |
| **角色** | 在一場戰役中，指的是四個參戰的彈珠。 |
| **彈珠** | 泛指各種角色持有的彈珠，也是戰鬥中的操作單位。玩家透過設定彈射方向與力道來操控彈珠。 |
| **AI 代管** | 當玩家不足 4 人時，由 AI 接手操控空缺席位的彈珠。 |

## 戰役與戰鬥

| 術語 | 定義 |
| :--- | :--- |
| **戰役** | 指進入關卡以後，直到挑戰成功或失敗的整個過程。包含了戰鬥、取得獎勵、遺物編織等等過程。戰役結束後會結算獎勵並回到城鎮。 |
| **戰鬥** | 指的就是單一關卡的戰鬥，會有一個明確的勝利條件，勝利後會獲得獎勵並繼續後續的戰役內容，失敗時該場戰役失敗。 |
| **回合** | 戰鬥以回合制進行，交替執行「玩家回合」與「敵方回合」。 |
| **彈射** | 玩家操作彈珠的基本行為——設定方向與力道，讓彈珠在場上移動、碰撞、觸發效果。 |
| **彈道** | 彈射時的預測路徑軌跡。 |

## 戰鬥數值

| 術語 | 定義 |
| :--- | :--- |
| **物理傷害** | 碰撞造成的傷害，以及特別指明為「物理傷害」的傷害。 |
| **魔法傷害** | 特別指明為「魔法傷害」的傷害。大多技能造成的傷害屬此類。 |
| **魔力** | 技能釋放所需的資源。可透過移動回魔、魔力殘渣、魔力泉等方式獲取。 |
| **技能（自動釋放）** | 戰鬥中，彈珠的技能在滿足條件時自動釋放，無需玩家手動操作。 |

## 遺物編織（Relic Weave）

| 術語 | 定義 |
| :--- | :--- |
| **遺物編織** | 每場戰鬥勝利後進入的配置階段（Relic Weave），透過串接遺物來客製化彈珠能力。屬隊伍共同決策。 |
| **遺物** | 可串接在彈珠上的強化物件，分為技能遺物、數字乘數遺物、轉換遺物三大類。 |
| **技能遺物** | 賦予彈珠技能的遺物，包含基礎技能遺物、主動技能遺物、被動技能遺物。 |
| **基礎技能遺物** | 每個彈珠內建的起始遺物，提供固定的能量輸出作為遺物鏈的起始動力源。 |
| **主動技能遺物** | 在戰鬥中獨立施放的技能遺物，具有魔力消耗與冷卻時間。 |
| **被動技能遺物** | 供電達標即永久生效的技能遺物，無需額外觸發。 |
| **數字乘數遺物** | 數值介於 0.5~2 之間的修正遺物，可鑲嵌至擴充孔以調整技能數值。 |
| **轉換遺物** | 改變能量流顏色屬性的中繼節點，代價是折損一定比例的傳遞能量。 |
| **能量** | 遺物串接的供電資源。前置遺物的能量輸出必須 ≥ 後方遺物的能量輸入需求，才能啟動。 |
| **擴充孔** | 遺物上的可用孔位，用於串接其他遺物或鑲嵌數字乘數遺物。孔洞具有顏色屬性（紅/綠/藍）。 |
| **Inactive** | 當遺物的能量輸入未達前置供電標準時的狀態。Inactive 的遺物及其後續所有遺物在戰鬥中不產生作用。 |
| **稀有度** | 遺物的品質等級：白 → 綠 → 藍 → 紫 → 橘。稀有度越高，擴充孔數越多。 |

## 遊戲結構

| 術語 | 定義 |
| :--- | :--- |
| **局內成長** | 單場戰役內的成長（遺物編織、打寶等），戰役結束後重置。 |
| **局外成長** | 跨戰役的永久成長（meta progression），讓玩家逐步解鎖新內容與能力。 |
| **打寶** | 戰鬥過程中產出裝備、素材、掉落物等，以「獲得與替換」驅動成長。 |
| **城鎮** | 戰役結束後回到的主要據點，進行局外成長、準備下一場戰役等活動。 |

## 世界觀與陣營

| 術語 | 定義 |
| :--- | :--- |
| **熊域（Bearrealm）** | 奧瑞亞大陸中央的高原密林之地，古老熊人部族的故鄉，遊戲的主要舞台。 |
| **永光碎片** | 聖殿之心碎裂後散落各地的光片，五大勢力爭奪的核心物件。 |
| **永光聖殿** | 熊域正心以白金與黑曜砌築的古老聖殿，因破曉之災而崩塌。 |
| **破曉之災** | 千年前奧瑞安帝國強行抽取聖殿之心導致的大災變，聖殿崩塌、碎片散落。 |
| **神聖聯盟（Sanctum）** | 代表色金/白。追求秩序，主張集中碎片於公共聖所共管。 |
| **烈焰軍團（Ember）** | 代表色紅/黑。信奉強者之道，主張以武力確保碎片安全。 |
| **野性誓約（Wild）** | 代表色綠/褐。熊域原住部族同盟，主張碎片應分散歸位自然。 |
| **以太學院（Aether）** | 代表色藍/白。來自浮空魔法環城，主張以研究與紀錄為先。 |
| **幽影議會（Umbral）** | 代表色紫/黑。目的成謎，於陰影中行動。 |
