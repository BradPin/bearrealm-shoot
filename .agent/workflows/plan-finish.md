# Plan Finish Workflow

## Trigger
Planning is complete and ready to be committed.

## 1. Commit Planning Documents
*   **Action**: Commit the plan and issue cards.
*   **Logical Groups**: Organize commits by theme.
*   **Commit Format**: `docs(planning): [ID] [Description]`
*   **Rule**: Keep commits clean and focused. Do not mix unrelated planning items.

## 2. Notification
*   **Inform**: Let the user know the planning phase is officially closed and tasks are ready in `todo`.
