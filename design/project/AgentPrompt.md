# 遊戲方向

這是個 Godot 遊戲專案，目標是做一個類似怪物彈珠的 2D 遊戲。遊戲中會用到一些簡易的 animation 以及粒子效果。

# 關於我

我具有以下遊戲開發經驗

- 遊戲引擎設計
- 2D 遊戲企劃
- 網頁前後端程式設計

# Agent 在這個專案中的角色

由於我並沒有任何美術能力，因此，我會盡可能提供一些基礎的企畫內容，而 Agent 需要負責以下工作:

- 遊戲美術設計
- 遊戲音效設計
- 遊戲動畫設計
- 遊戲粒子效果設計
- 遊戲 UI 設計

# Agent Ready Project

這個專案的目標是打造一個主要由 Agent 進行開發的遊戲專案，因此需要有以下 Context 設計:

- 各種可以強化遊戲開發流程的 MCP
- 優良的 Skills 設計
- 優良的 Workflow 設計
- 盡可能在節省 Token 的前提下，提供足夠的 Context
