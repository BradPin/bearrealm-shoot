# Master Art Prompt Vault

> This file contains the foundational style guidelines and prompt templates for all AI-generated image assets in this project.
> **AGENT DIRECTIVE**: Always inject the relevant style keywords from this vault into your image generation prompts.

## Global Style (Vibe)
**"Polished Japanese Fantasy Cartoon"**
- **Core Aesthetic**: High-quality 2D game asset, Japanese fantasy RPG illustration style. A perfect balance between clean cel-shading and soft painterly highlights. Characters have stout, chubby, and cozy proportions.
- **Shading & Texture**: **Soft cel-shading**. The boundaries between light and shadow are clear but have a slight softness/feathering. Smooth color transitions that maintain distinct color areas. 
- **Outlines**: **Clean, thin, dark-brown or colored outlines**. The linework is delicate and professional, providing structure without being heavy.
- **Color**: **Vibrant and warm**. High-quality digital illustration look with glowing magical accents.

## Prompt Templates

### 1. Character Portraits / Sprites
**Base Prompt**: `A high-quality 2D game asset of a [Subject], stout chubby proportions, Japanese fantasy RPG style, soft cel-shaded, clean thin dark outlines, polished digital illustration, vibrant warm colors, white background, masterpiece`

### 2. Characters in Marbles / Orbs (For Gameplay)
**Base Prompt**: `A high-quality 2D game asset of a [Subject] inside a glowing glassy sphere, soft cel-shaded, clean thin outlines, juicy highlights, Japanese fantasy RPG style, polished 2D game art, transparent background`

### 3. UI Elements (Icons / Frames)
**Base Prompt**: `A 2D game UI [Element Type] of a [Subject], Japanese fantasy RPG style, soft cel-shaded, thin clean outlines, chunky rounded forms, vibrant colors, transparent background`

## Negative Prompt (Mandatory)
`thick harsh black outlines, heavy painterly texture, messy lines, sketchy, realistic, 3d render, photographic, blurry, grainy, low saturation, extreme flat cel-shading`
