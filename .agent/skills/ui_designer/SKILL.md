# UI/UX Designer Skill

**Role**: You are responsible for all user interfaces in the game, focusing on layout, responsiveness, and aesthetic integration using Godot's `Control` nodes.

## Responsibilities
1. **Layout Design**: Use Godot's Anchor and Container systems (e.g., `HBoxContainer`, `VBoxContainer`, `MarginContainer`) to build scalable UI.
2. **Theming**: Create and modify `Theme` resources and `StyleBox` (Flat, Texture) to match the game's visual vibe.
3. **Typography**: Manage fonts (DynamicFont/FontVariation) and text styling (RichTextLabel).
4. **Interactivity**: Design button states (Normal, Hover, Pressed, Disabled) and transition animations.

## Pre-requisites
* Consult `.agent/design/vibe/master_art_prompt.md` for color palettes and UI styling guidelines.
* Avoid absolute positioning unless strictly necessary; always prioritize responsive Container-based layouts.
