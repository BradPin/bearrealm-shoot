# Project Vibe & Truth Source

## 1. Core Identity
*   **Title**: Bear Realm Shoot (Temporary)
*   **Theme**: "Bear Realm" (熊域爭鋒) - A polished arcade fantasy world.
*   **Tone**: Heroic, Energetic, Slightly Whimsical but not Silly.

## 2. Artistic Direction
**Style Goal**: Modern Vector Emblem / High-Fidelity 2D.

### Visual Pillars
1.  **Clean Lines**: Bold, clear outlines. Vector-like aesthetic.
2.  **Flat but Rich**: Flat colors with minimal, soft shading. No gritty textures.
3.  **Readability**: High contrast, suitable for fast-paced gameplay.

### Motifs
*   **Shapes**: Cirlce, Hexagons, Shields.
*   **Materials**: Stone, Gold, Magical Energy.
*   **Icons**: Bear Paws, Elemental Glyphs.

### Color Palette (Reference)
*   **Gold**: `#F5C242` (Accents, UI)
*   **Deep Brown**: `#4A3B32` (Primary, Backgrounds)
*   **Stone Grey**: `#9E9E9E` (Neutral)
*   **Elemental Colors**:
    *   Fire: Red/Orange
    *   Water: Cyan/Blue
    *   Wind: Green
    *   Earth: Brown/Amber
    *   Void: Purple
